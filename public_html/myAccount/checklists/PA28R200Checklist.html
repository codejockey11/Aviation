PRE-START

  • Preflight Complete
  • Passenger Briefing
  • Check Seats, Belts & Harnesses
  • Fuel Selector to Fullest Tank
  • Avionics Off
  • Electrical Switches Off
  • Check/Set Parking Brake
  • Check Circuit Breakers
  • Landing Gear Selector Down
  • Carb Heat Off

ENGINE START

  • Battery On
  • Alternator On
  • Fuel Pump On
  • Beacon On
  • Flaps Up
  • Propeller High RPM
  • Prime if cold engine start
  • Mixture Full
  • Throttle Open ½” Check For Fuel Flow
  • Mixture Cutoff
  • Prop Area CLEAR and so is REAR
  • Ignition Start and advance Mixture to Full
  • Throttle set 1000 RPM
  • Lean Mixture for Taxi
  • Check Oil Pressure
  • Carb Heat On and Off
  • Fuel Pump Off

PRE-TAXI
(Engine WarmUp)

  • Avionics On
  • Nav Lights On if needed
  • Transponder to Standby
  • Autopilot Off if installed
  • Check Annunciator Panel
  • Set Directional Gyro
  • Retrieve ATIS Note Weather
  • Set Altimeter
  • Set Avionics
  • Load Flight Plan
  • Call Clearance as needed
  • Call Ground at ramp spot with weather
  • Note Taxiways to Runway
  • Heading Indicator set to RWY
  • Check Turn Coordinator during taxi

RUN UP

  • Set Parking Break
  • Doors Closed and Latched
  • Flight Controls Free and Correct
  • Elevator Trim Set for Takeoff
  • Check Flight Instruments
  • Mixture to Rich
  • Throttle to 2000 RPM
  • Suction Gauge 5" ± .1"
  • Ammeter Alternator Off and On
  • Carb Heat On and Off
  • L&R Magnetos < 175 RPM drop
  • Prop Cycle RPM Drop
  • Prop Cycle Manifold Pressure Increase
  • Prop Cycle Oil Pressure drop
  • Mixture as needed
  • Throttle 1000 RPM
  • Set Friction Lock
  • Heading Indicator set to course heading
  • Short Field Flaps to 25°
  • Emergency Procedures rotate,low,500
  • Call Tower for departure/release direction
  • Note Time

TAKEOFF

  • Fuel Pump On
  • Lights On (lights)
  • Transponder to Alt (camera)
  • Throttle Full Power (action)
  • Monitor Engine
  • Rotate
  • Trim as needed
  • Short Field:
    • Climb Speed Vx
    • Retract Flaps when clear
  • Normal Field:
    • Climb Speed Vy
  • Tap Brakes
  • Gear Up
  • Contact Departure who,where,what

ENROUTE CLIMB

  • Airspeed maintain 500 fpm
  • Prop 2500 RPM
  • Throttle <= 25mp
  • Mixture as needed
  • Trim as needed

CRUISE

  • Throttle & Prop as Planned
  • Trim as needed
  • Mixture as needed
  • Landing Light Off
  • Fuel Pump Off
  • Check Directional Gyro to Compass
  • Fuel Selector as Required

LANDING

  • Contact Approach/Tower/CTAF
  • Fuel Pump On
  • Propeller Full
  • Mixture as needed
  • Trim as needed
  • Short Field:
    • Airspeed Downwind 100 Flaps 10°
    • Landing Gear Down
    • Airspeed Base 90 Flaps 25°
    • Airspeed Final 80 Flaps 40°
    • Break as needed
  • Normal Field:
    • Airspeed Downwind 100 Flaps 10°
    • Landing Gear Down
    • Airspeed Base 100 Flaps 25°
    • Airspeed Final 90 Flaps 40°

AFTER LANDING

  • Flaps Up
  • Landing Light Off
  • Exterior Lights as needed
  • Fuel Pump Off
  • Transponder to Standby
  • Lean Mixture for Taxi
  • Call Ground at spot to where
  • Note Taxiways to Ramp
  • Note Time

SHUTDOWN

  • Parking Brake Set
  • Avionics Off
  • Mixture Cutoff
  • Lights Off
  • Battery Off
  • Alternator Off
  • Ignition Switch Off
  • Close Flight Plan
